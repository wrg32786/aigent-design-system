# AIgent Landing Page Polish

Use this skill when a landing page already exists but needs to convert better and feel more premium.

## First Pass

Check:

1. What is the page selling?
2. Who already understands the terms?
3. What pain does the hero name?
4. What is the proof?
5. What action should the visitor take?

## Copy Rules

- Shorter is usually better.
- Put the outcome in the headline.
- Put mechanism in the subhead.
- Avoid jargon in the first viewport unless the audience already uses it.
- Do not say the same thing in subtext and cards.
- Use FOMO carefully: founder price, limited run, first access, before it gets crowded.

## CTA Rules

- Button label should name the product or action.
- Keep button dimensions stable when prices are added.
- Avoid cheap 3D effects.
- Add hover motion: subtle lift, glint, border light.
- Place guarantee/subscription clarity near the CTA.

## Premium Visual Rules

- Use fewer visual tricks, better executed.
- Do not use gradient blobs or decorative filler.
- Use real media, generated media, Spline, Three.js, or frame sequences for hero depth.
- Cards should have stable dimensions and enough breathing room.
- Text must never touch or overlap.

## Mobile Rules

- H1 must fit.
- CTA must be reachable without hunting.
- Sticky elements must not cover content.
- Font sizes must remain readable.
- Screenshots decide, not assumptions.

## Final Check

If the page feels impressive but unclear, simplify the copy.
If the page is clear but dull, improve the scene, type, and motion.
