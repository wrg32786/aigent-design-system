# Mobile QA Checklist

Run this before shipping any page built from this system.

## Viewports

- 360 x 740
- 390 x 844
- 430 x 932
- 768 x 1024

## Checks

- No horizontal scrolling.
- Header text appears before the section midpoint.
- Cards appear before the user has scrolled past the section title.
- Sticky nav does not cover content.
- Active nav chip centers itself.
- Tap targets are at least 40px high.
- Body copy is readable without zooming.
- Buttons do not resize when prices or labels change.
- External-link cues are part of the link, not dead icons.
- Background motion does not make text hard to read.

## Reduced Motion

Set reduced-motion in the browser or OS and confirm:

- Content is visible.
- Hover-only cues are not required.
- No essential information depends on animation.
