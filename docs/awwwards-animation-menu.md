## How To Use

Use the HTML menu to shortlist candidate effects by category, asset availability, visual example, and deck fit. Extract individual numbered folders from the archive only when an effect is selected for implementation.

For the AIgent sponsorship deck, prioritize:

- Page Transitions for click-to-page and chapter handoff moments.
- Text Animations for hero copy reveal, word fall, and type replacements.
- Scroll Animation for pinned scrub scenes, sticky fact migration, and staged side-entry text.
- WebGL / ThreeJS Effects for ambient hero depth and particle fields.

Keep heavyweight preview videos and nested code zips out of the design-system repo unless a selected effect becomes a reusable template.
